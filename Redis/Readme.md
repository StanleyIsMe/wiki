## Redis

Redis是一個開源的使用ANSI C語言編寫、遵守BSD協議、支持網絡、可基於內存亦可持久化的日誌型、Key-Value數據庫，並提供多種語言的API。

## How to install

env : ubuntu 14.04

`sudo apt-get update`

`sudo apt-get install redis-server`

## Basic Redis Shell

啟動redis
`redis-server`

進入redis terminal
`redis-cli`

設定key and value
`> set key "value"`

取得key值
`> get key`

刪除key
`> del key`
